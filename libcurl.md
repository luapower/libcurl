## `curl = require'libcurl'`

[LibCURL](http://curl.haxx.se/) binding.

## API

------------------------------------------------------- -------------------------------------------------------
__easy interface__
`curl.easy{options...} -> req`                          create a request
`req:perform() -> true|nil,err,ecode`                   perform the request
`req:free()`                                            free the request
`req:getinfo(opt) -> val`                               get info
`req:set(opt, val)`                                     set an option
`req:reset()`                                           reset all options to their default values
`req:recv(buf, bufsize, len) -> true|nil,err,errcode`   [receive raw data][curl_easy_recv]
`req:send(buf, bufsize, len) -> true|nil,err,errcode`   [send raw data][curl_easy_send]
`req.strerror(errcode) -> errmsg`                       look-up an error code
------------------------------------------------------- -------------------------------------------------------

[curl_easy_recv]: http://curl.haxx.se/libcurl/c/curl_easy_recv.html
[curl_easy_send]: http://curl.haxx.se/libcurl/c/curl_easy_send.html

## Easy interface

### `curl.easy{options...} -> req`

Create an request using the easy interface. Returns a table: the `CURL`
handle is in `req._curl`. Callbacks receive `req` as first arg.
FFI callback objects are freed automatically on `req:free()`.
